# からだにっき

からだにっきは、体重・夜の歩行・夜の主食量を毎日記録して、無理なく習慣化を続けるためのアプリです。  
達成度は `🥚 → 🐣 → 🐥 → 🐓` の成長で表示され、今日どれくらいできたかをやさしく確認できます。

## アプリ概要

- 体重を日ごとに保存できます
- 夜に歩けたかをチェックできます
- 夜の主食量を `ふつう / 半分 / なし` で記録できます
- 達成度に応じて動物が成長します
- 体重の推移と 7 日平均をグラフで確認できます
- 記録履歴を一覧で見返せます

## 起動方法

### 1. 依存関係をインストール

```bash
npm install
```

### 2. 開発サーバーを起動

```bash
npm run dev
```

ブラウザで表示されたローカル URL を開くと確認できます。

## ビルド方法

```bash
npm run build
```

Vite の標準構成なので、Vercel では以下の設定でそのままデプロイできます。

- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`
